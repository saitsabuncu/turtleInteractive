# Turtle Interactive

Python Turtle kütüphanesi kullanılarak geliştirilmiş etkileşimli bir çizim uygulamasıdır.

## Proje Hakkında

Bu proje, klavye ve kullanıcı etkileşimlerini kullanarak Turtle Graphics ile çizim yapılmasını amaçlamaktadır.

## Özellikler

* Klavye ile kontrol
* Gerçek zamanlı çizim
* Turtle Graphics kullanımı
* Olay tabanlı programlama örneği

## Kullanılan Teknolojiler

* Python 3
* Turtle Graphics

## Amaç

Bu proje, Python öğrenme sürecimde kullanıcı etkileşimleri, klavye olayları ve grafik programlama konularını uygulamalı olarak öğrenmek amacıyla geliştirilmiştir.
